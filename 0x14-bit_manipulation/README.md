# Bit Manupulation
## Bitwise Operators
* ```&``` - and   
* ```|``` - or   
* ```~``` - negation    
* ```^``` - xor  
* ```>>``` - right shift  
* ```>> ``` - left shift
A = 00000001  
B = 10001001  
```A&B = 00000001```  
```A|B = 10001001```  
```A^B = 10001000```  
```~A = 11111110```  
```A>>1 = 00000000```  
```B>>1 = 01000100```  
```A<<3 = 00001000```
